    How to Play: Integer Sci-Damath

    Objective: Finish the game with the lowest total score. Lower score wins!

    Setup

    • Red pieces (top): [-9, 6, -1, 4, 0, -3, 10, -7, -11, 8, -5, 2]
    • Blue pieces (bottom): [2, -5, 8, -11, -7, 10, -3, 0, 4, -1, 6, -9]
    • Pieces start on dark squares only.

    Movement

    • Regular piece: moves diagonally forward one square.
    • Dama (King): promotes when reaching the opponent's back row. Can move/capture any distance diagonally.

    Capturing

    • Captures are mandatory. If you can capture, you must.
    • Jump over an opponent’s piece to an empty square beyond it.
    • After a capture, if another capture is possible with the same piece, you must continue (multi-jump).
    • Mayor Dama Rule: If multiple capture options exist, you must choose the sequence that captures the most pieces. If tied, prefer starting with a Dama.

    Scoring a Capture

    When capturing, apply the mathematical operator (+, −, ×, ÷) on the destination square: 

    Result = (Your Value) [operator] (Opponent's Value)

    Then apply multipliers:

    • ×2 if either capturing or captured piece is a Dama.
    • ×4 if both are Damas.

    The result is added to your running score.

    End of Game

    • Game ends when:
      • A player has no legal moves.
      • All of one player’s pieces are captured.
      • Surrender or mutual agreement to finish.
    • Remaining pieces are tallied:
      • Regular piece: add its value once.
      • Dama: add its value twice.
    • Final winner = player with the LOWER total score.

    Tip: A negative score is key for winning!
